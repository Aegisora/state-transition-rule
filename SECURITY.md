# Security Policy

## Reporting a Vulnerability

If you find a vulnerability, please do not disclose it publicly.

Contact me via:

* Email: [ara.imamutdinov@gmail.com](mailto:ara.imamutdinov@gmail.com)
* Or direct message (if you know how to reach me)

Please include:

* what the issue is
* steps to reproduce

## Response

I will try to respond within a few days and fix the issue as soon as possible.

## Disclosure

Please allow time for a fix before public disclosure.

Thank you 🙌
